## Requirements

* \> php 5.5
* composer
* npm
* bower

## Setup

* git clone https://github.com/luknei/github-issues.git
* cd github-issues
* composer install
* cd public
* php -S localhost:8000
* visit http://localhost:8000/

## Features

* Login/logout to github
* List repository issues
* Read issue and issue comments
* Update issue
* Close issue
